プライバシーポリシー

サービス名：SaidLog　／　制定日：2026年6月16日　／　運営者：個人

運営者（以下「当方」）は、SaidLog（以下「本サービス」）におけるユーザーの個人情報の取り扱いについて、以下のとおり定めます。

1. 収集する情報

  • メールアドレスおよびハッシュ化されたパスワード（Supabase Auth による認証情報）
  • 文字起こしテキスト（ログイン時のみ、Supabase データベースに保存）
  • 音声ファイル（処理のため一時的に Supabase Storage に保存し、処理完了後に自動削除）
  • ログイン日時・アカウント登録日時

2. 外部サービスへの送信

  • 音声データ：Groq, Inc. に送信し、音声認識処理（Whisper v3 Turbo）を行います。データは米国の Google Cloud Platform 上で処理され、推論データはデフォルトで保持されません。
  • 文字起こしテキスト：Anthropic 社の Claude API に送信し、AI による要約処理を行います（要約機能使用時のみ）。

3. データ保持期間

  • 音声ファイル：処理完了後に即時削除します。削除失敗時の安全網として、Supabase Storage の使用量に応じて最大30日で自動削除します。
  • 文字起こしテキスト：ユーザーが削除するまで保持します（表示は直近30件）。
  • アカウント情報：ユーザーがアカウントを削除するまで保持します。

4. ユーザーの権利

ユーザーはアプリ内のアカウント削除機能から、すべてのデータ（文字起こしテキスト・音声ファイル残存分・アカウント情報）を削除できます。

5. 第三者提供

上記「外部サービスへの送信」に記載した処理委託を除き、ユーザーの情報を第三者に提供することはありません。

6. Cookie・アクセス解析

本サービスは現時点でアクセス解析ツールおよびトラッキング Cookie を使用していません。

7. お問い合わせ

個人情報の取り扱いに関するお問い合わせは、下記までご連絡ください。
メール：support@solidaily.com

8. ポリシーの変更

本ポリシーは必要に応じて改定することがあります。重要な変更がある場合は、本ページおよびアプリ内でお知らせします。

Privacy Policy

Service: SaidLog　／　Effective date: June 16, 2026　／　Operator: Individual

The operator of SaidLog ("we" or "us") sets out the following policy regarding the handling of user information in this service.

1. Information We Collect

  • Email address and hashed password (authentication credentials managed by Supabase Auth)
  • Transcription text (stored in Supabase database only when the user is signed in)
  • Audio files (temporarily stored in Supabase Storage for processing; automatically deleted after processing)
  • Login timestamps and account registration date

2. Data Sent to Third-Party Services

  • Audio data: Sent to Groq, Inc. for speech recognition (Whisper v3 Turbo). Data is processed on Google Cloud Platform in the United States and is not retained by default.
  • Transcription text: Sent to Anthropic's Claude API for AI-generated summarization (only when the summary feature is used).

3. Data Retention

  • Audio files: Deleted immediately after processing. As a safety net for deletion failures, files are automatically deleted within a maximum of 30 days based on Supabase Storage usage.
  • Transcription text: Retained until deleted by the user (up to the 30 most recent entries are displayed).
  • Account information: Retained until the user deletes their account.

4. User Rights

Users can delete all their data — including transcription text, any remaining audio files, and account information — using the account deletion feature available within the app.

5. Disclosure to Third Parties

We do not share user information with third parties except for the processing services described in Section 2 above.

6. Cookies and Analytics

This service does not currently use any analytics tools or tracking cookies.

8. Changes to This Policy
